## インジェクション系のまとめ

|  脆弱性  |  インターフェース  |  悪用手口  | データの終端  |
| ---- | ---- |---- |---- |
|  XSS  |  HTML  | JavaScriptなどの注入  | `<` `"` など  |
|  SQLi  |  SQL  | SQL 命令の注入 | `'` など  |
|  RCE  |  シェルスクリプト  | コマンドの注入  | `;` `\|` など  |
|  HTTPヘッダ・インジェクション  |  HTTP  | HTTP レスポンスヘッダの注入  | 改行 |
|  メールヘッダ・インジェクション  |  sendmail コマンド  | メールヘッダ、本文注入・改変 | 改行 |
